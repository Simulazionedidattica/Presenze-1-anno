VERSIONE STABILE SENZA GITHUB

Questa versione deve essere pubblicata interamente dentro Google Apps Script.

1. Nel progetto Apps Script sostituisci tutto Codice.gs con Code.gs.
2. Premi + accanto a File.
3. Scegli HTML.
4. Chiama il file esattamente: Index
5. Incolla nel file Index il contenuto di Index.html.
6. Salva.
7. Esegui il deployment > Gestisci deployment.
8. Matita > Nuova versione.
9. Esegui come: Me.
10. Chi ha accesso: Chiunque.
11. Distribuisci.
12. Apri sul cellulare il link /exec mostrato da Google.

Non usare più il link GitHub Pages.
Il foglio di destinazione e la scheda sono già configurati.
